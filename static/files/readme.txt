100LAR-WEB-DISK

Данный ресурс является копией других дисков (Google drive, Яндекс диск), но у него есть несколько фишек, которых нет у других - он может запускаться на любом сервере и имеет открытый исходный код.
Место на этом хранилище зависит от сервера.

github : https://github.com/4LAR/100LAR-WEB-DISK/tree/master
